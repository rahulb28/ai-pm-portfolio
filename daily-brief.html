Personal AI Assistant

Rahul's AI
Daily Brief

A personal AI assistant I built to start every morning with a clear, prioritized overview of my day. It pulls together my calendar, emails, job search pipeline, kids' school updates, coursework deadlines, and upcoming webinars — then emails me a beautifully formatted HTML brief before I even open my laptop.

What it does
📅
Calendar Intelligence
Reads Google Calendar, detects meeting gaps, and surfaces the best deep focus window of the day.
🎯
Job Search Pipeline
Scans ATS emails (Greenhouse, Lever, Workday), classifies each as Interview / Received / Rejection / Recruiter, and tracks active applications.
📬
Smart Inbox Triage
Filters out newsletters and promotions. Surfaces only real-person emails with urgent deadlines, RSVPs, or action items.
🏫
School & Family
Monitors teacher emails, permission slips, no-school days, and school events — so nothing slips through.
📚
Coursework & Webinars
Tracks assignment deadlines, webinar RSVPs, certification milestones, and any learning commitments in the next 7 days.
🚀
Focus Plan
Synthesizes everything into a ranked top-3 priority list and the best focus window — so mornings start with clarity, not chaos.
How it's built

Built as a Claude Code skill that runs inside my local Claude Code session with Gmail + Google Calendar MCP tools. For fully automated morning delivery, a GitHub Actions workflow runs at 7 AM daily using the Anthropic API (claude-sonnet-4-6) with Gmail API + Calendar API + a custom tool-use agent loop. The HTML brief is sent directly to my inbox via EmailJS.

Claude Sonnet 4.6 Anthropic API Claude Code Gmail API Google Calendar API GitHub Actions EmailJS MCP Tools Tool-use Agent Loop Python
