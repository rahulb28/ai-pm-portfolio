## Deploy to GitHub Pages (5 minutes)

1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → `main` → `/ (root)`**
4. Your portfolio will be live at `https://[your-username].github.io/ai-pm-portfolio`

> For the AI PRD Generator to call the Claude API, the app uses a direct browser-side API call. This works in local/demo environments. For production, proxy through a backend.
